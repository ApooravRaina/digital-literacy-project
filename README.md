# Digital Literacy Project

## Overview
This project is part of the Digital Literacy course. It includes multiple tasks designed to improve understanding of digital tools, online safety, and professional presence.

## Tasks Completed
- Digital Literacy Infographic (Task 1)
- Student Digital Portfolio (Task 2)
- Coding Practice and Google Form Quiz (Task 3)
- Email Etiquette and Social Media Do’s & Don’ts (Task 4)
- Cybercrime Case Study and Prevention Checklist (Task 5)

## Tools Used
- Canva (for infographic design)
- GitHub (for project submission)
- LinkedIn, Kaggle (for portfolio)
- HackerRank (for coding practice)
- Google Forms (for quiz creation)

## Conclusion
This project helped in understanding the importance of digital literacy, safe online practices, and building a professional digital presence.
